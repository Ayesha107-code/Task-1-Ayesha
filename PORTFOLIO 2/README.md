# Internship Task 1: Personal Portfolio Website

Welcome to my very first internship project! This repository contains a personal portfolio website designed and developed from scratch as part of my frontend web development training. It is built using clean, structural HTML5 and custom CSS3.

The goal of this task was to create a modern, visually appealing, and organized single-page portfolio that highlights my developer journey, skill sets, and early projects.

---

## 🚀 Purpose & Project Scope

As an aspiring Frontend Developer, this project served as a practical foundation to implement core web concepts. The primary objectives achieved in this task include:
- Structuring a complete webpage using clean **Semantic HTML**.
- Implementing a modern, dark-themed user interface using native **CSS Styling**.
- Organizing multi-column card items using **CSS Grid** and **Flexbox** techniques.
- Building interactive micro-animations (button hovers, input focus rings, smooth scrolling transitions).

---

## ✨ Features Implemented

- **Sticky Navigation Bar:** A fixed navigation section at the top featuring a modern background blur effect (`backdrop-filter`) for accessible page jumping.
- **Hero Intro Section:** A welcoming section introducing myself, my stack focus, and immediate call-to-action buttons.
- **Interactive Skill Badges:** Distinct layout chips highlighting core tools like HTML, CSS, JavaScript, and React.
- **Responsive Project Gallery:** Displays my learning milestones (such as the *Event Hub* and *Coursera Clone*) in custom grid cards with individual hover transition effects.
- **Clean Contact Interface:** A structured form layout optimized with clear input fields, active state indicators, and explicit labels.

---

## 🛠️ Tech Stack Used

- **HTML5:** For standard semantic layout structure.
- **CSS3:** For customized presentation rules, colors, transitions, and structural layouts (Flexbox/Grid).
